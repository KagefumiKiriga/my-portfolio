# 請求書自動化 実践記録
## 作成日：2026年5月7日

---

## 今日作ったものの全体像

フォームに入力するだけで以下が全自動になる仕組みを作った。

入力 → 記録 → 請求書作成 → PDF変換 → ドライブ保存 → メール送信

---

## Step 1：Googleフォームの作成

### 作ったもの
「請求書発行フォーム」というGoogleフォーム

### フォームの項目
1. 発行日（日付形式）
2. 宛先会社名（記述式）
3. 案件名（記述式）
4. 単価（記述式）
5. 数量（記述式）
6. 備考（記述式）

### ポイント
- Googleドライブの「請求書自動化_練習」フォルダに作成
- 公開設定を「リンクを知っている全員」に設定

---

## Step 2：スプレッドシートとの連携

### やったこと
フォームの回答タブからスプレッドシートを新規作成

### 結果
フォームに入力するたびに以下の列に自動記録される
- A列：タイムスタンプ
- B列：発行日
- C列：宛先会社名
- D列：案件名
- E列：単価
- F列：数量
- G列：備考

---

## Step 3：GASの設定

### GASとは？
Google Apps Scriptの略。
Googleのツール同士を自動でつなぐプログラム。
MacBookで例えると「Automator（自動化アプリ）」みたいなもの。

### トリガーの設定
- 実行する関数：onFormSubmit
- イベントのソース：スプレッドシートから
- イベントの種類：フォーム送信時

→ フォームが送信されるたびに自動でGASが動く

---

## Step 4：GASのコード（全体）

### コードの役割
1. フォームデータを取得する
2. 合計金額を自動計算する
3. 請求書シートを作成する
4. PDFに変換する
5. Googleドライブに保存する
6. Gmailで送信する

### コード
function onFormSubmit(e) {
var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
var lastRow = sheet.getLastRow();
var date = sheet.getRange(lastRow, 2).getValue();
var company = sheet.getRange(lastRow, 3).getValue();
var project = sheet.getRange(lastRow, 4).getValue();
var price = sheet.getRange(lastRow, 5).getValue();
var quantity = sheet.getRange(lastRow, 6).getValue();
var memo = sheet.getRange(lastRow, 7).getValue();
var total = price * quantity;
var tax = Math.round(total * 0.1);
var totalWithTax = total + tax;
var ss = SpreadsheetApp.getActiveSpreadsheet();
var invoiceSheet = ss.getSheetByName("請求書") || ss.insertSheet("請求書");
invoiceSheet.clear();
invoiceSheet.getRange("A1").setValue("請求書");
invoiceSheet.getRange("A3").setValue("発行日：");
invoiceSheet.getRange("B3").setValue(date);
invoiceSheet.getRange("A4").setValue("宛先：");
invoiceSheet.getRange("B4").setValue(company + " 御中");
invoiceSheet.getRange("A6").setValue("案件名");
invoiceSheet.getRange("B6").setValue("単価");
invoiceSheet.getRange("C6").setValue("数量");
invoiceSheet.getRange("D6").setValue("金額");
invoiceSheet.getRange("A7").setValue(project);
invoiceSheet.getRange("B7").setValue(price);
invoiceSheet.getRange("C7").setValue(quantity);
invoiceSheet.getRange("D7").setValue(total);
invoiceSheet.getRange("A9").setValue("小計");
invoiceSheet.getRange("D9").setValue(total);
invoiceSheet.getRange("A10").setValue("消費税（10%）");
invoiceSheet.getRange("D10").setValue(tax);
invoiceSheet.getRange("A11").setValue("合計（税込）");
invoiceSheet.getRange("D11").setValue(totalWithTax);
invoiceSheet.getRange("A13").setValue("備考：");
invoiceSheet.getRange("B13").setValue(memo);
var ssId = ss.getId();
var sheetId = invoiceSheet.getSheetId();
var url = "https://docs.google.com/spreadsheets/d/" + ssId + "/export?format=pdf&gid=" + sheetId;
var token = ScriptApp.getOAuthToken();
var response = UrlFetchApp.fetch(url, {headers: {"Authorization": "Bearer " + token}});
var pdfBlob = response.getBlob().setName(company + "請求書" + date + ".pdf");
var folder = DriveApp.getRootFolder();
var file = folder.createFile(pdfBlob);
var email = "自分のGmailアドレス";
var subject = company + " 様 請求書送付のご案内";
var body = company + " 御中\n\nお世話になっております。\n請求書をPDFにてお送りいたします。\n\n案件名：" + project + "\n合計金額：" + totalWithTax + "円（税込）\n\nご確認のほどよろしくお願いいたします。";
GmailApp.sendEmail(email, subject, body, {attachments: [pdfBlob]});
Logger.log("メール送信完了：" + file.getUrl());
}
